# OpenDog Project Exchange

Central directory for OpenDog usage reports, suggestions, feedback templates, and cross-project operating notes.

Detailed usage rules and archived-asset index live in [QUICKSTART.md](../../QUICKSTART.md#project-exchange-reports).

## Layout

- `templates/` - reusable report templates for other projects.
- `reports/<project>/` - imported or submitted project-specific OpenDog reports.
- `issues/INDEX.md` - shared issue index for problems visible across projects.

## Rule

Keep one-to-one project communication under `reports/<project>/`; use `issues/INDEX.md` only for cross-project shared issues.
